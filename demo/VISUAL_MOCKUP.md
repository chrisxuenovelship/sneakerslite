## User Flow

### Discovery
1. User lands on homepage
2. Clicks **"Blog"** in header navigation
3. Lands on `/blog` (listing page)
4. Sees grid of articles (demo article visible)

### Engagement
5. Clicks article card → Navigates to `/blog/real-vs-fake-...`
6. Reads full content (headings, bullets, paragraphs)
7. Sees featured image at top
8. Scrolls to bottom → Sees CTA: "Ready to Authenticate?"
9. Clicks **"Get Started"** button → Goes to `/contact` page

### Repeat
10. User can click **"Back to Blog"** → Returns to listing
11. Future articles appear automatically as published

## Interactive Elements

### Hover States
- **Article cards** → Shadow darkens, slight lift effect
- **Header links** → Text color changes to brand-cyan
- **Buttons** → Background darkens, cursor changes to pointer
- **Links** → Underline on hover

### Active States
- **Current page link** → Underlined (currently on /blog)
- **Mobile menu** → Background highlight on selected item
