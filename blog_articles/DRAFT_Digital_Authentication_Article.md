## The Dashboard: What Our Metrics Actually Measure

When a pair goes through our authentication system, we measure **8 specific dimensions**:

### 1. **Size Tag ID** ✓ VERIFIED
- **What it is:** Machine-readable verification that the size tag format matches Nike's official specifications
- **Why it matters:** Fake factories struggle with tag printing; the format is encoded in manufacturing databases
- **How we check:** Database lookup against Nike's official GTIN records + physical size tag OCR
- **False positive rate:** <1% (very reliable)

### 2. **Size Tag Font & Spacing** (98.2%)
- **What it is:** OCR analysis of font weight, character spacing, and kerning on the size/care tag
- **Why it matters:** Nike uses specific typefaces; counterfeits use cheap substitutes that have measurable differences
- **How we check:** AI pixel-level analysis + comparison to 5,000+ verified examples
- **Why 98.2% (not 100%):** Material aging and lighting variations; margin of error built in
- **Seller value:** This single metric catches ~60% of budget fakes

### 3. **Color Accuracy** (97.6%)
- **What it is:** Spectral analysis of shoe colors against Nike's official color specifications
- **Why it matters:** Counterfeiters source cheaper dyes; these have measurable color shifts under controlled lighting
- **How we check:** Photo analysis using color profiling + comparison to authenticated pairs in same colorway/year
- **Seller value:** Catches material sourcing fakes (usually $50–$150 counterfeits)

### 4. **Outer Stitching** (96.1%)
- **What it is:** Stitch count, spacing, and pattern recognition on the shoe exterior
- **Why it matters:** Professional stitching equipment creates consistent patterns; cheap factories use older machines
- **How we check:** AI stitch detection + database comparison (stitching patterns vary by model/year/factory)
- **Seller value:** Catches mid-tier fakes; high-tier fakes often pass this

### 5. **Logo Precision** (92.4%)
- **What it is:** Vector analysis of the Jumpman/Wings/Swoosh logos (positioning, angle, proportions)
- **Why it matters:** Logo placement is locked in the design; fakes often have measurable positioning differences
- **How we check:** Point-cloud mapping + comparison to 10,000+ verified examples per model
- **Seller value:** Catches 70% of mid-tier fakes; combined with other metrics, very reliable

### 6. **Insoles** (93.9%)
- **What it is:** Insole material texture, color, text clarity, and font analysis
- **Why it matters:** Insole manufacturing is hard to replicate; texture variations are hard to fake at scale
- **How we check:** High-res image analysis of insole surface + OCR of insole text + material database
- **Seller value:** Very strong indicator; high-tier fakes sometimes fail here

### 7. **Midsole Stitching** (97.7%)
- **What it is:** Stitch count and pattern on the midsole (outsole junction)
- **Why it matters:** Midsole assembly is done by hand; stitch patterns vary slightly per pair (authentic) vs. identical per batch (fake)
- **How we check:** Computer vision + pattern recognition + database comparison
- **Seller value:** One of the most reliable checks; catches high-tier fakes

### 8. **Box/Packaging** (93%)
- **What it is:** Box material durability, label print quality, and packaging consistency
- **Why it matters:** Packaging materials change yearly; fakes often use outdated or incorrect materials
- **How we check:** Material analysis + label OCR + comparison to authenticated pairs from same release date
- **Seller value:** Supplementary check; useful for full-lot verification

## Verification Checks: What Gets Flagged

Our system runs **6 core verification checks** that appear in every authentication report:

### ✓ Authentic Materials Detected
- **What it checks:** Database comparison of material composition (leather type, mesh weave, rubber formulation)
- **Why it matters:** Counterfeit leather is typically PU or cheap cow hide; texture analysis reveals it
- **How:** Material database + image texture analysis
- **False positive rate:** 3–5% (some rare authentic batches use alternative suppliers)

### ✓ Logo Shape Alignment Pass
- **What it checks:** Jumpman/Wings/Swoosh logos are within tolerance of official vectors
- **Why it matters:** Logo placement varies <2mm on authentics; fakes often have >5mm variance
- **How:** Point-cloud geometry + vector comparison
- **False positive rate:** <2% (very reliable)

### ✓ Stitching Pattern and Count Verified
- **What it checks:** Stitch count on uppers, insoles, and midsole matches authenticated samples
- **Why it matters:** Stitch count is hard to replicate; varies by model/year/factory but is measurable
- **How:** Computer vision + stitch detection + database comparison
- **False positive rate:** 4–6% (authentic variation exists, especially older pairs)

### ✓ Serial Number Checks Verified
- **What it checks:** Size tag serial number exists and is authentic (for 2010+ models)
- **Why it matters:** Fakes often have invalid or missing serial numbers; we can check authenticity codes
- **How:** Connects to Nike's manufacturing database (partnership pending in 2026)
- **False positive rate:** 2–3% (stolen/recycled tags are edge cases)

### ✓ Font and Spacing with Tolerance
- **What it checks:** All text (care tags, size tags, insole text) matches official fonts within manufacturing tolerance
- **Why it matters:** Font replication is surprisingly hard; even small variations compound
- **How:** OCR + font database + tolerance band (±0.5–2mm depending on region/era)
- **False positive rate:** 5–8% (older pairs had larger tolerances; regional variations exist)

### ✓ Counterfeit Markers: None
- **What it checks:** Absence of known counterfeit indicators (bad glue, visible seams, logo bleeding, etc.)
- **Why it matters:** High-tier fakes don't fail on obvious markers; this check catches budget fakes
- **How:** Image analysis + database of 3,000+ known counterfeit defects
- **False positive rate:** <1% (very specific checks; low noise)
